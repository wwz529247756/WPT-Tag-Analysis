## Notes on Test Coverage

1. **Comprehensive Reflection Testing**: All standard HTML5 elements have reflection tests in their respective category files.

2. **Deprecated Elements**: Only 5 of 28 deprecated elements have specific reflection tests (`dir`, `font`, `frame`, `frameset`, `marquee`).

3. **ARIA Testing**: Separate test files exist for ARIA attribute reflection, which applies to many HTML elements.

4. **IDL Testing**: The `idlharness.https.html` file provides comprehensive interface testing for all HTML elements.

5. **Special Attribute Testing**:
   - `usvstring-reflection.https.html` tests USVString handling for URL attributes
   - `access-key-label.html` tests the `accessKeyLabel` property
   - `self-origin.sub.html` tests the `self.origin` property for iframes

6. **Historical Elements**: The `historical.html` file tests some historical elements not in the standard lists.
